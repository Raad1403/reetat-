# منصة الذكاء العقاري للمطوّرين (MVP)

هذا المشروع هو نسخة أولية (MVP) لمنصة ويب موجهة للمطوّرين العقاريين في السوق السعودي والخليجي.

## التشغيل

1. تثبيت الحزم:

```bash
npm install
```

2. تشغيل بيئة التطوير:

```bash
npm run dev
```

ثم افتح المتصفح على: `http://localhost:3000`

## ملاحظات

- الواجهة عربية بالكامل ومهيأة لاتجاه RTL.
- صفحات جاهزة:
  - `/` صفحة الهبوط.
  - `/auth/register` تسجيل مطوّر جديد.
  - `/auth/login` تسجيل الدخول.
  - `/dashboard` لوحة تحكم تجريبية مع مشاريع وهمية.
  - `/projects/new` نموذج إنشاء مشروع تطوير عقاري جديد.
  - `/projects/[id]` صفحة عرض النتائج (حاليًا بيانات ثابتة تجريبية مع محاكاة الفرق بين المجاني والمدفوع).
- لم يتم بعد ربط النظام بقاعدة بيانات، ذكاء اصطناعي، أو بوابة دفع ميسر. سيتم تجهيز هذه الأجزاء في مراحل قادمة.
